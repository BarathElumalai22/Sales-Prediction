# Sales Prediction using Machine Learning

## Overview

This project implements a Supervised Machine Learning Regression model to predict product sales based on advertising expenditures across different media channels.

The model analyzes advertising budgets allocated to TV, Radio, and Newspaper campaigns and predicts the resulting sales.

---

## Dataset Information

The dataset contains advertising expenditure data and corresponding sales figures.

### Features

| Feature | Description |
|----------|-------------|
| TV | Advertising budget spent on TV |
| Radio | Advertising budget spent on Radio |
| Newspaper | Advertising budget spent on Newspaper |

### Target Variable

| Variable | Description |
|-----------|-------------|
| Sales | Product sales generated |
